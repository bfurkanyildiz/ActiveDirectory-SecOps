# Kaynaklar ve Teknik Referanslar (ChatGPT Deep Research)

Bu araştırma sürecinde, Active Directory sıkılaştırma ve güvenlik standartları için aşağıdaki kurumsal ve teknik kaynaklar analiz edilmiştir:

## Kurumsal ve Resmi Dokümantasyonlar
* **Microsoft Learn:** [Active Directory Security Best Practices & Tier Model](https://learn.microsoft.com/en-us/windows-server/identity/ad-ds/plan/security-best-practices/best-practices-for-securing-active-directory) (Tiered Administration ve GPO temelleri için kullanıldı)
* **Microsoft Tech Community:** [Protecting Tier 0 the Modern Way](https://techcommunity.microsoft.com/) (Yönetici hesaplarının izolasyonu stratejisi)
* **Microsoft Security Guidance:** [Mitigate Critical Threats to AD DS in 2025](https://microsoft.com) (Kerberoasting ve Unconstrained Delegation çözümleri)

## Güvenlik Standartları ve Analiz Raporları
* **CIS Benchmarks (via Tenable):** [Windows Server 2022 v4.0.0 L1 DC](https://www.tenable.com) (14 karakterlik parola zorunluluğu ve sistem sıkılaştırma ayarları)
* **ExtraHop:** [2025 Security Predictions: Active Directory Still a Target](https://extrahop.com) (AD zafiyetlerinin güncel tehdit ortamındaki analizi)
* **Verizon:** [Data Breach Investigations Report (DBIR)](https://thehackernews.com) (Kimlik bilgilerinin ihlallerdeki %88'lik rolü verisi)

## Teknik Uygulama ve Siber Güvenlik Blogları
* **Specops Software:** [AD Security Best Practices for 2025](https://specopssoft.com) (GPO tasarımı, OU yapısı ve felaket kurtarma planları)
* **Compass Security:** [Finding AD Attack Paths using BloodHound](https://blog.compass-security.com) (Saldırı yollarının görselleştirilmesi ve ACL hataları)
* **The Hacker News:** [Active Directory Under Siege](https://thehackernews.com) (Kritik altyapı güvenliği ve zayıf parola riskleri)
* **Semperis:** [Group Policy Abuse Explained](https://semperis.com) (GPO suistimali ve zararlı yazılım yayma riskleri)
